Time Stacking - Control Law Reconfiguration

The learner predicts false positives and negatives in fault-free and turbulence scenarios, respectively. Nevertheless, such a situation is undesirable to use, as it is, to coordinate the control law reconfiguration system. That is, even scarce false positives within a true negative time window could suggest changing the control law in an intermittent and undesirable way. Therefore, an additional system is required to avoid such situations.

A basic implementation is proposed here, using only the previously trained decision tree in a time window of one second, using the median value of the counting of failure / no-failure prediction.
